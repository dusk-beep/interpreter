01_OUTPUT

`parayu` / print

Print numbers, strings, and interpolated values like `{a}`.

namaskaram
    a=10
    parayu a
    parayu "Thank you"
nanni

02_INPUT

`eduku` / input

Use cast syntax like `(int)eduku` and provide browser input one line at a time.

namaskaram
    a=(int)eduku "no1:"
    b=(int)eduku "no2:"
    sum=a+b
    parayu "sum:{sum}"
nanni

03_CONDITIONS

`anengil`, `anengil enkil`, `allengil`

Single-line blocks are allowed, and curly braces are needed for multi-line blocks.

namaskaram
    a=10
    anengil(a>8)
    {
        parayu "{a}>8"
    }
    anengil enkil(a<0)
    {
        parayu "{a}<0"
    }
    allengil
        parayu "{a}<8"
nanni

04_FUNCTION

`pravarthanam` / function

Create reusable logic and return values with `thirike kodku`.

pravarthanam kootuka(a,b)
{
    thirike kodku (a+b)
}

namaskaram
    a=10
    b=20
    c=kootuka(a,b)
    parayu c
nanni

05_RECURSION

Recursive function

Functions can call themselves, so recursion works for factorial-style problems.

pravarthanam fact(n)
{
    anengil(n==0)
        thirike kodku 1
    allengil
        thirike kodku n*fact(n-1)
}

namaskaram
    parayu fact(5)
nanni

06_STRUCT

`ghatana` / struct

Define a custom structure once and access members with dot notation.

ghatana student
{
    per="lavanya"
    vays=10
};

namaskaram
    ghatana student s;
    parayu s.per
    parayu s.vays
nanni

07_FOR_LOOP

`avarthikuka` / for

C-style loop syntax is supported, including `i++` update expressions.

namaskaram
    avarthikuka(i=1;i<4;i++)
        parayu "hello{i}"
nanni

08_ARRAYS

Arrays and datatypes

Use array literals, integer and float values, chars, strings, and indexed access.

namaskaram
    markukal[]={1,2,3,4}
    vyaktata[]={1.1,2.2}
    avarthikuka(i=0;i<4;i++)
        parayu markukal[i]
    parayu vyaktata[0]
    parayu vyaktata[1]
nanni
